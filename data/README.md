## Dataset 1 — CQ500 (primary, head/skull CT)

- **Source:** Academic Torrents — https://academictorrents.com/details/47e9d8aab761e75fd0a81982fa62bddf3a173831
- **Torrent file:** `~/Downloads/qure.headct.study-47e9d8aab761e75fd0a81982fa62bddf3a173831.torrent`
- **Size:** ~28.66 GB
- **Volumes:** 491 head CT scans (CQ500-CT-0 through CQ500-CT-490)
- **Format:** DICOM series, one ZIP per patient
- **License:** CC BY-NC-SA 4.0 (non-commercial, attribution, share-alike)
- **Download date:** 2026-04-20

### Download command used
```bash
transmission-cli '/Users/jasminavulovic/Downloads/qure.headct.study-47e9d8aab761e75fd0a81982fa62bddf3a173831.torrent' --download-dir 'data/raw/cq500/'
```

### Unzip + convert one volume (run as each .zip completes)
```bash
# Unzip a single volume
unzip data/raw/cq500/qure.headct.study/CQ500-CT-0.zip -d data/raw/cq500/qure.headct.study/

# Convert to NIfTI
source venv/bin/activate
python scripts/dicom_to_nifti.py \
    --input  data/raw/cq500/qure.headct.study/CQ500-CT-0/ \
    --output data/raw/cq500_nifti/

# Visualise
python scripts/visualise_volume.py data/raw/cq500_nifti/CQ500-CT-0.nii.gz
```

### Batch unzip + convert (run after all ZIPs are downloaded)
```bash
for zip in data/raw/cq500/qure.headct.study/*.zip; do
    unzip -q "$zip" -d data/raw/cq500/qure.headct.study/
done
python scripts/dicom_to_nifti.py \
    --input  data/raw/cq500/qure.headct.study/ \
    --output data/raw/cq500_nifti/
```

### Integrity check
```bash
ls data/raw/cq500_nifti/*.nii.gz | wc -l   # expect 491
```
